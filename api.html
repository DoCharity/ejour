API文档

使用API前，您需要知道：

最后更新 ： 2016年4月25日

易游的API目前版本为1，此版本源代码也是开源的，具体地址位于我们的 GitHub 主页上。

1获取方式
API的地址位于 https://api.ejour.co/v1/api.php ，获取方式可采取Post的方式。格式形如：
https://api.ejour.co/v1/api.php?spot=yzsrt&id=1
这将返回一个Json数据，包括该景区的概述信息和展品的详细信息供您的应用程序调用。

2Spot(必须)
Spot是每个景区的唯一标识符，最少三个字符。您可以在管理员后台申请创建新的景区，标识符创建后不能轻易修改。
spot=yzsrt
3ID(必须)
ID是每个景区内展品的唯一标识符，从1开始编号。您可以在管理员后台添加或删除展品。
id=1
4Info(已弃用)
已弃用
5Format(可选)
Format指定了输出的格式，默认为json，可不填。
format=json
